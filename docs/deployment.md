# Deployment

## Distribution model

This project is distributed as source script and optional ZIP artifact generated by CI/local build.

## Release flow

1. Update `CHANGELOG.md` using Keep a Changelog format.
2. Ensure CI is green on `main`.
3. Create release tag using SemVer.
4. Attach `dist/windows-tuneup-script.zip` artifact to GitHub Release.

## Operational requirements

- Target machine: Windows 10/11 (best effort for 7/8/8.1)
- Runtime: Windows Script Host (`cscript`/`wscript`)
- Privilege: Administrative rights required for maintenance operations
